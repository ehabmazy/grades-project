# 📚 سجل الدرجات — مدرسة الشهيد حمزة

تطبيق ويب متكامل لإدارة درجات الطلاب، يعمل بالكامل في المتصفح دون الحاجة إلى خادم أو قاعدة بيانات خارجية.

---

## 🌐 العرض المباشر

> يمكن رفع المشروع على **GitHub Pages** والوصول إليه مباشرةً عبر الرابط:
> `https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 📊 **سجل الدرجات** | إدخال وتعديل درجات الطلاب لكل فصل دراسي |
| 📅 **الجدول الأسبوعي** | تتبع الحضور والغياب أسبوعياً |
| 🔔 **نظام الإشعارات** | تنبيهات عند بدء/انتهاء الفترات |
| 📝 **صفحة الإملاء** | إدخال الدرجات عبر الكتابة النصية |
| 📈 **الإحصائيات** | تقارير وتحليلات لأداء الطلاب |
| 🗓️ **الجدول الدراسي** | إدارة جدول الفصول والفترات |
| 📤 **تصدير Excel** | تصدير البيانات إلى ملف `.xlsx` |
| 💾 **حفظ تلقائي** | جميع البيانات تُحفظ في `localStorage` |
| 🔐 **تسجيل الدخول** | نظام حماية بسيط بكلمة مرور |
| 📱 **متجاوب** | يعمل على الهاتف والحاسب |

## 🚀 كيفية الاستخدام

### الطريقة الأولى: فتح مباشرة (بدون إنترنت)
```
افتح ملف index.html مباشرةً في أي متصفح
```

### الطريقة الثانية: GitHub Pages
1. ارفع المشروع إلى GitHub
2. اذهب إلى **Settings → Pages**
3. اختر الفرع `main` والمجلد `/ (root)`
4. احفظ — سيُنشر الموقع خلال دقيقة

---

## 🗂️ هيكل الملفات

```
grades-project/

├── LICENSE             ← رخصة الاستخدام

```

---

## ⚙️ المتطلبات التقنية

- متصفح حديث (Chrome / Firefox / Edge / Safari)
- لا يحتاج Node.js أو Python أو أي خادم
- مكتبة **SheetJS** (`xlsx`) تُحمَّل تلقائياً من CDN

---

## 💡 ملاحظات مهمة

- **البيانات محلية:** كل البيانات مخزنة في `localStorage` في المتصفح — لا تُرفع إلى الإنترنت
- **النسخ الاحتياطي:** استخدم زر "تصدير Excel الكامل" من صفحة الإعدادات بانتظام
- **مسح البيانات:** زر "مسح نهائي" في الإعدادات يحذف كل شيء بشكل لا رجعة فيه
